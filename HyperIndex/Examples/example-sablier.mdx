The following indexers serve as exceptional reference implementations for the Sablier protocol, showcasing professional development practices and efficient multi-chain data processing.

## Overview

[Sablier](https://sablier.com/) is a token streaming protocol that enables real-time finance on the blockchain, allowing tokens to be streamed continuously over time. These official Sablier indexers track streaming activity across 18 different EVM-compatible chains, providing comprehensive data through a unified GraphQL API.

## Professional Indexer Suite

Sablier maintains three specialized indexers, each targeting a specific part of their protocol:

<CardGroup cols={3}>
  <Card title="Lockup Indexer" icon="lock" href="https://github.com/sablier-labs/subgraphs/tree/main/apps/lockup-envio">
    Tracks the core Sablier lockup contracts for token streaming with fixed durations. Handles stream creation, cancellation, and withdrawal events. Used primarily for vesting functionality.
  </Card>
  
  <Card title="Flow Indexer" icon="water" href="https://github.com/sablier-labs/subgraphs/tree/main/apps/flow-envio">
    Monitors advanced streaming functionality with dynamic flow rates. Captures stream modifications and batch operations. Powers the payments side of Sablier.
  </Card>
  
  <Card title="Merkle Indexer" icon="diagram-project" href="https://github.com/sablier-labs/subgraphs/tree/main/apps/merkle-envio">
    Tracks Merkle distribution system for efficient batch stream creation. Provides data about batch creations and claims. Used for Airstreams and Instant Airdrops.
  </Card>
</CardGroup>
