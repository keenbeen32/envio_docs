# Envio Docs

Latest Envio documentation built with Mintlify.

## Prerequisites

- Node.js (>=20.0)
- [Mint CLI](https://www.npmjs.com/package/mint) - `npm i -g mint` (to be able to preview locally)
- pnpm

## Commands

- `pnpm dev` - Preview locally
- `pnpm update-endpoints` - Update supported networks
